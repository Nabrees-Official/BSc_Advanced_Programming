# Ocean View Resort Room Reservation System

This system allows staff to manage room reservations, calculate bills, and print invoices for guests. It is a Java EE application with a MySQL database backend.

## Features:
- User authentication
- Add, edit, view, and delete reservations
- Calculate guest bills based on room rates
- Responsive user interface

## Setup:
1. Clone the repository
2. Set up a MySQL database and import `database.sql`
3. Run the Java EE project using an appropriate IDE (e.g., IntelliJ IDEA, Eclipse)

## Usage:
- The system supports adding new reservations, viewing existing ones, and calculating bills for guests.
